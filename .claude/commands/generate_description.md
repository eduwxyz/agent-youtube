Leia o arquivo $ARGUMENTS.arquivo e analise a transcrição do vídeo.

Com base no conteúdo, gere uma descrição otimizada para YouTube seguindo estas diretrizes:

## Estrutura da descrição:

### 1. Gancho inicial (primeiras 2-3 linhas)
- As primeiras 150 caracteres aparecem na busca - use-os com sabedoria
- Resuma o valor principal do vídeo
- Inclua a palavra-chave principal naturalmente

### 2. Corpo da descrição
- Expanda sobre o que o espectador vai aprender/ver
- Use parágrafos curtos (2-3 linhas)
- Inclua 3-5 palavras-chave secundárias de forma natural
- Adicione timestamps dos principais momentos do vídeo (se aplicável)

### 3. Call-to-Action (CTA)
- Peça para se inscrever no canal
- Incentive likes e comentários
- Mencione vídeos relacionados (placeholders)

### 4. Links e recursos
- Seção para links mencionados no vídeo
- **OBRIGATÓRIO** - Sempre incluir as redes sociais do criador:
  - LinkedIn: https://www.linkedin.com/in/eduardo-machado-141835192/
  - GitHub: https://github.com/eduwxyz

### 5. Hashtags
- 3-5 hashtags relevantes no final
- Mix de hashtags populares e específicas do nicho

## Regras de SEO:

1. **Palavras-chave**: Identifique e use naturalmente as principais palavras-chave do conteúdo
2. **Tamanho ideal**: Entre 200-500 palavras (descrições mais longas rankeiam melhor)
3. **Evite**:
   - Keyword stuffing (repetição excessiva)
   - Links suspeitos ou spam
   - Texto copiado de outros lugares
4. **Inclua**:
   - Perguntas para estimular comentários
   - Emojis estratégicos para destacar seções (sem exagerar)

## Formato de saída:

Gere APENAS 1 descrição completa, pronta para copiar e colar.

**IMPORTANTE**: A descrição DEVE incluir os links do LinkedIn e GitHub nas redes sociais.

Gere a descrição de forma profissional e engajante, mantendo a autenticidade do criador.
